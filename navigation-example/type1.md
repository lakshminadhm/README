# Combining Links with Horizontal Rules
Separate navigation links with horizontal rules (---) for clarity. Prev at the top and Next at the end

## Example Code
```markdown
[Previous: DUMMY PREV](./dummyPrev.md)

---

# Page Title

Content goes here...

---

[Next: DUMMY NEXT](./dummyNext.md)
```

## Output

[Previous: DUMMY PREV](./dummyPrev.md)

---

# Page Title

Content goes here...

---

[Next: DUMMY NEXT](./dummyNext.md)
